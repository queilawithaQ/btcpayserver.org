Поддержать проект BTCPay

BTCPay Server - процессор криптовалютных платежей с открытым исходным кодом и самостоятельным хостингом. Это безопасно, приватно, устойчиво к цензуре и бесплатно.

Corporate Donations

Компании и организации, желающие поддержать проект, могут это сделать на сайте BTCPay Server Foundation
BTCPay Server Foundation  

Частные пожертвования

Если вы не являетесь представителем какой-либо компании или организации, вы можете напрямую материально поддержать кого-либо из участников проекта. Пожертвования осуществляются на личных BTCPay-серверах разработчиков, не относящихся к BTCPay Server Foundation.
